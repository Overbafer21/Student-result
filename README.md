# Student Result Management System (Java Swing)

Desktop-приложение на **Java Swing** для управления результатами студентов.

## Что умеет
- Авторизация администратора (`admin / 1234`).
- Ввод данных студента и оценок по 3 предметам.
- Автоматический расчет:
  - общей суммы,
  - процента,
  - статуса (**PASS/FAIL**).
- Сохранение результатов в файл `results.txt`.
- Просмотр ранее сохраненных записей.

## Что изменено относительно базового варианта
- Полностью обновлен визуальный стиль (не 1 в 1):
  - экран входа с карточным дизайном,
  - отдельная dashboard-страница,
  - двухколоночный интерфейс (форма + область результатов),
  - улучшенные шрифты/цвета/отступы.
- Улучшена безопасность ввода пароля: используется `JPasswordField`.
- Добавлена валидация диапазона оценок (0–100).

## Запуск
Требуется JDK 11+.

```bash
javac StudentResultManagement.java
java StudentResultManagement
```

## Структура
- `StudentResultManagement.java` — основной файл приложения.
- `results.txt` — создается автоматически после первого сохранения результата.
