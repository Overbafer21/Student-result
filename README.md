# Система учёта результатов студентов (Java Swing)

Небольшое desktop-приложение на **Java Swing** для расчёта, сохранения и просмотра учебных результатов студентов.

## Возможности
- Авторизация администратора (`admin / 1234`).
- Ввод данных студента и оценок по 3 предметам.
- Автоматический расчёт:
  - общей суммы баллов,
  - процента,
  - итогового статуса (**СДАЛ / НЕ СДАЛ**).
- Сохранение результата в файл `results.txt`.
- Просмотр всех ранее сохранённых записей.

## Изменения дизайна
Интерфейс специально переработан и не повторяет исходный шаблон 1 в 1:
- экран входа в виде карточки;
- отдельная панель (dashboard) после авторизации;
- двухколоночная компоновка: форма ввода + область вывода результатов;
- более аккуратные отступы, цвета и шрифты.

## Запуск
Требуется **JDK 11+**.

```bash
javac StudentResultManagement.java
java StudentResultManagement
```

## Структура проекта
- `StudentResultManagement.java` — основной исходный код приложения.
- `results.txt` — файл, который создаётся автоматически при первом сохранении результата.
